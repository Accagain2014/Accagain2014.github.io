《Machine Learning》-01 Introduction

　　回家之前这段时间争取把Andrew Ng的机器学习视频看完吧，作为入门，做工程挺没意思的。看了机器学习的一些应用觉得挺有意思的，好好琢磨一下吧。慢慢的尝试，然后找到自己的方向。


Definition:

Arthur Samuel (1959). Machine Learning: Field of study that gives computers the ability to learn without being explicitly programmed.

Tom Mitchell (1998). Well-posed Learning Problem: A computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured by P, improves with experience E.*

第一种定义：机器学习——通过给计算机学习能力，而没有明确地编程的领域。
第二种定义：适定的学习问题——对于任务T，用评测标准P，性能能够根据训练集E提升的计算机程序。

机器学习算法:

主要：
　　　　Supervised Learning [监督学习]
　　　　Unsupervised Learning [无监督学习]
其他：
　　　　Reinforcement learning[加强学习],recommender systems[推荐系统]

Supervised Learning:[监督学习]

    'right answer' given：训练集是正确的


　　Regression:[回归]　predict continuous[连续] values output
　　Classification:[分类]　discrete[离散] valued ouput

Unsupervised Learning:[无监督学习/聚类算法]

总结和比较：

无人机自主学习、分离噪声等例子好有趣。
