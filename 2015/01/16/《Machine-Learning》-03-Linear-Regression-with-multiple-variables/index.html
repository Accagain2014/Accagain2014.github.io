梯度下降算法实用技巧：特征归一化

如果变量X范围相差很大，可以通过变量缩放，归一化。使各个X都相对差不多，这样可以加速收敛过程。

Mean normalization：均值归一化 X=(X-U)/S (U为平均值，S为Max-Min)

Gradient descent in practice II:Learning rate

　梯度下降算法实用技巧：关于步长



　　关于α的选择：太小了，会减小收敛的速度；太大了，则在每次迭代时，J不会减少。可以根据迭代次数与J值的函数关系，适当做些评估，然后改变α的值：如果随着迭代次数的增多，J上升则减小α，直到J减小幅度达到精度。因为很小的α肯定没问题，所以还可以开始用比较小的α，然后每次乘以3倍：0.01,0.03,0.1，0.3,1……..直到达到比较好的效果。
