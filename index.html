Python 07 正则表达式

正则表达式

  • 正则表达式是一种用来匹配字符串的强有力的武器。它的设计思想是用一种描述性的语言来给字符串定义一个规则，凡是符合规则的字符串，就称为“匹配”了，否则该字符串就是不合法的。
  • \d 匹配一个数字
  • \w 匹配一个字母或数字
  • * 表示任意个字符(包括0个)

Python 06 进程和线程

进程和线程

  • 对于操作系统来说，一个任务就是一个进程(Process)，例如打开浏览器、打开记事本等。有些进程不只干一件事，比如Word，他可以同时进行打字、拼写检查、打印等事情。在一个进程内部，要同时干多件事，就需要同时运行多个“子任务”，进程内的这些子任务称为线程(Thread)。
  • 由于每个进程至少干一件事，所以一个进程至少有一个线程。
  • 多任务实现方式：多进程模式、多线程模式、多进程加多线程模式
  • 同时执行多个任务通常各任务之间需要相互通信和协调。Python既支持多线程也支持多进程。

Python 03 模块 && 面向对象编程

模块

  • 在Python中，一个.py文件就称之为一个模块(Module)
  • 使用模块好处
    　　1、提高了代码可维护性
    　　2、避免函数名和变量名冲突，主要在编写模块时，尽量不要和内置函数名字冲突
  • 为了避免模块名字冲突，Python引入了按目录来组织模块的方法，称为包(Package)
  • 每个包目录下面必须要有个__init__.py文件，这个文件是必须的，否则，Python就把这个目录当成普通目录，而不是一个包
  • __init__.py本身就是一个模块，而它的模块名就是包名
